**Hosting (GitHub Pages)**

This website is hosted via GitHub Pages (GitHub, Inc./GitHub B.V.). When you access this site, GitHub automatically processes technical data, in particular the IP address of the accessing device, in order to deliver the site and to operate and secure its infrastructure. Legal basis: Art. 6(1)(f) GDPR (legitimate interest in the secure operation of the website). More information: [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

**Formula rendering (MathJax via jsDelivr)**

On pages containing mathematical formulas, the MathJax library is loaded from the jsDelivr content delivery network to render formulas in the browser. This transmits the IP address of the accessing device to jsDelivr. Legal basis: Art. 6(1)(f) GDPR. More information: [jsDelivr Privacy Policy](https://www.jsdelivr.com/terms/privacy-policy-jsdelivr-net).

**No cookies, no tracking**

This website itself does not set cookies and does not use any analytics or tracking services. No personal data is collected via forms, as no such functionality is offered.
